# ReinforcementL-GridWorld-Traps-main
Gridworld Traps: Compare RL algorithms (Dynamic Programming, Monte Carlo, SARSA, Q-Learning) in a 12×12 grid with obstacles. Model-based &amp; model-free approaches, policy/value visualizations, flexible config. Educational tool for studying convergence, exploration, and trap avoidance in discrete MDPs. 🎯🤖
